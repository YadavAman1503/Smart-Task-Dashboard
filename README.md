# Smart Task Dashboard

Advanced Flask-based Smart Task Management System with AI Analytics and Real-Time Monitoring

---

# 📌 Features

## ✅ Authentication
- User Login
- User Registration
- Session Management

## ✅ Smart Task Management
- Add Tasks
- Complete Tasks
- Delete Tasks
- Pending Tasks
- Completed Tasks

## ✅ Real-Time Notifications
- Browser Notifications
- SocketIO Live Updates
- Toast Alerts

## ✅ Advanced Analytics
- Pie Visualization
- Bar Analytics
- 3D Productivity Graphs
- AI Performance Analysis
- Live Task Statistics

## ✅ Dashboard
- Dark Mode UI
- Interactive Graphs
- Responsive Design
- Live Productivity Tracking

## ✅ Backend Features
- Flask REST APIs
- PostgreSQL Database
- SQLAlchemy ORM
- Pandas Analytics
- NumPy Calculations
- WebSocket Integration
